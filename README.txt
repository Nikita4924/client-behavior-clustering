#  DECENTRATHON 3.0 — Поведенческая сегментация клиентов

Проект выполняет **кластеризацию клиентов (держателей карт)** по их транзакционной активности с помощью алгоритмов **HDBSCAN** и **UMAP**, без обучения с учителем.

---

Основные этапы

1.  Загрузка и очистка данных
2.  Feature Engineering:
   - Средние и суммарные траты
   - Кол-во уникальных магазинов
   - MCC категория
   - Продолжительность активности
3. Кластеризация без учителя (HDBSCAN)
4. Визуализация кластеров (UMAP)
5. Сохранение результатов (`.csv`, `.parquet`, `.png`)
